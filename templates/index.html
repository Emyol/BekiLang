💅 The BekiLang Cheat Sheet 💅

1. Data Types & Declarations

  • borta (Integer) - Whole numbers
  • mema (Float) - Decimal numbers
  • chika (String) - Text in quotes ""
  • charot (Char) - Single character ''
  • truch (Boolean) - korik (True) / wiz (False)

2. Operators & Delimiters

  • ay = Assignment (var ay 5)
  • periodt / ganern = Statement Delimiter (;)
  • parinig = Print / Output Console

3. Arithmetic & Logic

  • Math: dagdag (+), bawas (-)
    times (*), divide (/)
  • Compare: mas_tarush (>), mas_chaka (<)
    parehas (==), wit_parehas (!=)
  • Boolean: at_saka (AND), o_kaya (OR)

4. Control Flow (Loops & Ifs)

  • kunwari [cond] eh_di [cond] = If / Else statement
  • gorabel [cond] { ... } = While Loop
