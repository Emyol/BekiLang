💅 The BekiLang Cheat Sheet 💅

1. Data Types

  • borta (Integer) - Whole numbers
  • mema (Float) - Decimals
  • chika (String) - Text ""
  • charot (Char) - Single ''
  • truch (Bool) - korik / wiz

2. Operators

  • ay = Assign (x ay 5)
  • periodt / ganern = ;
  • parinig = Print

3. Math & Logic

  • dagdag (+), bawas (-)
  • times (*), divide (/)
  • mas_tarush (>), mas_chaka (<)
  • parehas (==), wit_parehas (!=)
  • at_saka (AND), o_kaya (OR)

4. Control Flow

  • kunwari [cond] eh_di [cond]
  • (If / Else)
  • gorabel [cond] { ... }
  • (While Loop)
